Runtime routing (sequence)

sequenceDiagram
  actor Caller
  participant R as router.classify
  participant M as mermaid (lazy)
  participant N as native reader+renderer
  participant D as diagnostics
  Caller->>R: raw DSL (detectType)
  alt flowchart
    R-->>Caller: sync themed render (no mermaid)
  else sequence/class/state
    R->>M: render to SVG
    M-->>N: mermaid SVG
    N-->>Caller: native themed + interactive
  else other type
    R->>M: fallback render
    alt clean (pie/browser)
      M-->>Caller: mermaid SVG
    else degenerate headless
      M->>D: fallback-render-unavailable
      D-->>Caller: clear diagnostic, no broken output
    end
  end
