gogo diagrams — interactive-editing

Offline viewer · mermaid is vendored with the plugin (no network, no CLI) · regenerate via the gogo-mermaid skill.

As-built flow (v0.4.0): node / group / anchor edits → perimeter re-anchor + auto-contain → re-route → persist / export
