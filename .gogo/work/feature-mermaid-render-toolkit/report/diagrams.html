As-built modules & types

classDiagram
  class DiagramModel {
    +direction: Direction
    +nodes: DiagramNode[]
    +edges: DiagramEdge[]
    +subgraphs: Subgraph[]
    +classDefs: Map~string, StyleDef~
    +diagnostics: Diagnostic[]
  }
  class PositionedModel {
    +nodes: PositionedNode[]
    +edges: RoutedEdge[]
    +subgraphs: PositionedSubgraph[]
    +bounds: Rect
  }
  class RoutedEdge {
    +from: string
    +to: string
    +path: string
    +waypoints: Point[]
    +label?: string
  }
  class Theme {
    +tokens: TokenSet
    +edgeStyle: EdgeStyle
  }
  class Parser {
    +parse(dsl, ParseOptions) DiagramModel
  }

    +layout(model, LayoutOptions) PositionedModel
  }

    +routeEdge(a, b, dir, style, waypoints) RoutedEdge
    +elbowThrough() ~orthogonal~
    +roundedPath() ~curved~
    +snapWaypoints()
  }
  class SvgRenderer {
    +renderSvg(input, opts) string
  }
  class AsciiRenderer {
    +renderAscii() string
    +renderMarkdown() string
  }
  class DomRuntime {
    +mount(el, opts) RuntimeHandle
    +vnmRuntime (serializable)
  }
  class HtmlExport {
    +renderHtml(input, opts) string
  }
  class PngExport {
    +renderPng(input, opts) Uint8Array
  }
  class VeryNiceMermaidElement

  Parser --> DiagramModel : produces
  Layout --> DiagramModel : reads
  Layout --> PositionedModel : produces
  Layout ..> Geometry : routes edges
  PositionedModel *-- RoutedEdge
  SvgRenderer ..> PositionedModel

  AsciiRenderer ..> PositionedModel
  DomRuntime ..> PositionedModel
  DomRuntime ..> Theme
  DomRuntime ..> Geometry : re-routes on drag
  HtmlExport ..> DomRuntime : serializes vnmRuntime
  PngExport ..> SvgRenderer : rasterizes via resvg
  VeryNiceMermaidElement ..> DomRuntime : mount()

Interactive runtime: drag → live re-route → persist

sequenceDiagram
  actor User
  participant El as mount() / element
  participant RT as vnmRuntime
  participant Geo as geometry
  participant LS as localStorage

  Note over El,RT: initial render
  El->>RT: mount(el, {model, theme})
  RT->>Geo: routeEdge(a,b,dir,style,waypoints)
  Geo-->>RT: RoutedEdge.path (elbow/curved through waypoints)
  RT-->>User: node cards + SVG edge layer + minimap

  Note over User,LS: drag to reorganize (live re-route)
  User->>RT: pointerdown on node card
  loop pointermove
    RT->>RT: update node position (÷ zoom)
    RT->>Geo: re-route connected edges
    Geo-->>RT: new paths (stay anchored to borders)
    RT-->>User: edges follow the node
  end
  User->>RT: pointerup
  RT->>LS: persist layout (debounced)

  Note over User,RT: toolbar & viewport
  User->>RT: click fit / zoom (guarded from pan capture)
  RT-->>User: transform updated
  User->>RT: reload page
  RT->>LS: read saved layout
  LS-->>RT: positions
  RT-->>User: arrangement restored
